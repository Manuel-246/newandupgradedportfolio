# Manuel Binoy Portfolio Website

## Overview

A modern, responsive personal portfolio website showcasing my skills, projects, education, and career goals as an aspiring Data Scientist and Full-Stack Developer. The website features an elegant UI/UX design with dark/light theme support, smooth animations, project filtering, and responsive layouts for all devices.

## Features

* Modern and responsive design
* Dark and Light theme toggle
* Interactive hero section with typing animation
* About Me section with tab navigation
* Technical skills showcase
* Project portfolio with filtering and search functionality
* Contact section
* Smooth scrolling navigation
* Mobile-friendly interface
* Glassmorphism UI elements
* Animated visual effects

## Technologies Used
